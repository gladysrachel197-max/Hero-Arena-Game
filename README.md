# Hero-Arena-Game(Java)
A turn-based battle arena gam developed as part of my Computing course

## Features
- **OOP Concept:** Uses Classes, Inheritance, and Encapsulation
- **Game Logic:** Turn-based combat with health points and attack stats

## Tech Stack
- **Language:** Java
- **IDE:** Netbeans


  ## How to Run
  1. Clone the repo
  2. Compile 'Main.java'
  3. Run the application in your terminal
